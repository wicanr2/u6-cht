# 聖者之書 1992 版 vs 本專案 譯名校對報告

**參考來源**：電腦玩家雜誌 1992/5 出版《創世紀聖者之書 特別版》(DDSC-J-00007)
作者：高文麟 (序) / 蘇炫榮 (前言、各章節)，83 頁

## 整體觀察

聖者之書 1992 版採「**保留英文專有名詞 + 翻譯概念詞**」策略，與本專案
「**全面中文化**」路線顯著不同。例如：

- 書中：「`Lord British` 的城堡」「`Iolo` 是吟遊詩人」「`Mondain` 巫師」
- 我們：「不列顛王的城堡」「尤洛是吟遊詩人」「巫師蒙丹」

兩種風格各有道理：書版方便玩家對照英文版遊戲；本專案目標是純中文沉浸。
**人物 / 地名 / 城市名 不採納**書版（保留我們中文音譯）。

## 採納的修正（要 apply）

**只一條核心修正**：

1. **守則之書 → 知識寶典**（39 處）— 採納聖者之書權威譯
2. **終極智慧守則之書 → 終極智慧之寶典**（2 處）

加 review hint：

3. 「勇氣」當 Valor 用時建議改「勇敢」— 但 NPC 對話中「勇氣」一字常意指 courage（一般概念非 virtue label），逐句辨識難。**保留為手工 review item**，不機械替換。
